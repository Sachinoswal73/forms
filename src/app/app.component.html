Forms

- Though,, Form is already provided by html. Angular gives some extra features like creating object of all the form fields, error-handling & validations.
- We can use Forms module using template driven approach ( code in html template ) or Reactive approach ( code in ts file ) .
- to use the ngModel we need to import Forms module. we need to give the 'name' to the fields which we're going to use in Form module.
- As angular is SPA, we cannot add multiple Submit button in one page. So, ANgular manage this by providing us ngSubmit, ngForm.
- This helps in reducing the work of backend developer.
- We can use patch value & setValue method to update the form objects in both template & reactive approach.
- Difference between Template Driven Approach & Reactive Approach is that We can use asynchronous validators in Reactive Form. Eg.Email validation if any email id is in existence. We can create our own custom validators, custom async validators etc. We can use array as well.
